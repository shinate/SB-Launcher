# SD_FM

FARM 专用，圣斗可以没有顾虑的使用防御姿势

- 武神力量
- 贤哲封印
  - 魔力保护 [数值,默认:200] 可设置魔力保护，低于此值时不施放
  - 自动拉怪 [true/false,默认:true] 默认会从范围内找最远的一个目标施放
- 正义封印
  - 范围保护 [数值,默认:50] 可设置范围，范围内无目标则不施放
  - 魔力保护 [数值,默认:200] 可设置魔力保护，低于此值时不施放
- 圣洁封印
  - 范围保护 [数值,默认:50] 可设置范围，范围内无目标则不施放
  - 魔力保护 [数值,默认:200] 可设置魔力保护，低于此值时不施放
- 尼斯神符
  - 失血保护 [数值,默认:300] 可设置损失多少血时施放
  - 照顾队友 [true/false,默认:false] 默认只管自己，如果要照顾队友可开启
